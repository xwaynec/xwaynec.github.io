Baltic Sea 3 Countries - Poland Zakopane

Dec 26th, 2013

Zakopane 是波蘭的城鎮，位於波蘭南部，由小波蘭省負責管轄，始建於十七世紀，面積84平方公里，最高點海拔高度1,126米，是熱門的旅遊地點。主要是爬上山去可看海洋之眼(Morskie Oko)。Zakopane 坐落在塔特拉山(Tatra)和 Gubałowki 山之間的山谷, 海拔約在 800-1000m 之間,市中心就在 KrupówkStreet(行人徒步街)與 KościuszkoStreet 之間。 Zakopane 是位於克拉科夫南方的一個波蘭原住民小鎮,在這不僅可以體會波蘭傳統文化,還 可以欣賞到最美的湖光山色景緻。

我們從華沙坐車到Zakopane就先殺去hostel。 Willa Marma B&B。

油超貴…

這個馬路我想開賽車都可以…

路上其實是要去Krupówk Street步行街，就跟墾丁大街很像XD，但是實在是太餓了，走在路上就被炸物招牌吸引…

進去這間就是噩夢的開始，差點出不來啊…

看似親切的阿豆仔，每個都超會喝，被灌到差點出不來。印象很深的一句就是: This is fucking Poland，不過真的半滴酒錢都沒出就是了 XD

逃出來先買冰激淋壓壓驚…

接下來沿路就是市集

到處都有賣冰淇淋，歐洲人真的很愛吃冰淇淋，三小國到處都可見。你可以找不到任何一間商店，但是可以看到賣冰淇淋的攤販….

搭愣~

我絕對不是故意的…

慢慢人有變多

這邊是一個墓園，雖然我不信教，不過尊重死者我就沒多拍了。

Restauracja Owczarnia ul Krupowki 26, Zakopane 34-500 , Poland 晃一圈之後肚子又餓啦，在外面看到這畫面只好乖乖進去消費了。其實這間沒有很貴 科科

Menu作成一個斧頭的形狀?!

登愣 again~ 一整個就是膽固醇報表，超肥超好吃。整趟旅程下來我竟然反瘦2kg~

接著就先回宿舍睡了，隔天一大早要搭Gubalowka纜車。

隔天路上隨便拍

大概走了3km?! 然後被一群中學生超過，來到纜車的base station，起大霧…不過昨天到hostel之後，已經知道看不到海洋之眼(Morskie Oko)，所以才改換Gubałowki行程。

真的是阿宅，這樣也要拍…

到達山線之後，真的是冷翻了我的媽啊…

這個應該是滑雪的

根據外國人的臉孔，這個應該是中學左右

其實在這邊搭吊籃(lift)下山，再走 2KM 回到市區，正好繞一圈，不過吊籃沒開啊阿啊啊~ anyway 山上起大霧沒有待很久就下山，草草吃個東西就去趕車了，冬天這邊是滑雪勝地，因為便宜。

Alaska - Aurora in Fairbanks & Polarbear in Barrow

Dec 1st, 2013

畢業之後一直都想去阿拉斯加看極光和北極熊，但是礙於鬼島的研發替代役只能出國8天，只好延後到服完役期，且剛好2012~2013是極光極大期。這次去阿拉斯加總共10天, 其中去Barrow看北極熊是9/24，當天從Fairbanks來回Barrow；剩下九天都在Fairbanks。

行前準備

9月的阿拉斯加其實不會冷到靠背，晚上頂多-5℃ ，把去合歡山賞雪的裝備帶一帶就好了…(Barrow除外，這真是可怕的地方，穿兩件褲子還是會從腳底冷上來)，以下是冬天的準備(參考):

  • 頭部：保暖的帽子（毛線帽或頭套）與圍巾（耳罩可視個人需要準備)護膚乳、護唇膏、太陽眼鏡。若有需要也請帶足量的個人適用常備藥物。

  • 手部與腳部：厚手套（操作相機時最好還要有薄手套）、棉襪與毛襪（多一層絲襪也能保暖）、登山鞋（底面能止滑，儘量選用防水與保暖的材質，若有多一點的空間還可塞暖暖包）。

  • 穿著：以「多層式」穿法為宜，因為車子與室內都有暖氣供應，最好能穿脫方便。衣服：長衛生衣褲、毛衣、棉質衣服、厚襯衫、厚羽絨衣外套（能防水最好）。褲子：棉質長褲、牛仔褲，外面若能再罩一件雙層的防風防水褲最好。

Activemerchant Allpay Plugin for Taiwan Payment

Nov 28th, 2013

ActiveMerchantAllpay

This plugin is an active_merchant patch forAllpay(歐付寶) online payment in Taiwan. Now it supports Credit card(信用卡), ATM(虛擬ATM) and CVS(超商繳費).

Installation

Add this line to your application’s Gemfile:

gem 'activemerchant'
gem 'active_merchant_allpay', '>=0.1.0'

And then execute:

$ bundle install

Or install it yourself as:

$ gem install activemerchant
$ gem install active_merchant_allpay
