Warsaw Uprising Museum 華沙起義博物館

官方網站

華沙起義博物館，紀念1943年二次大戰時，為推翻納粹對華沙的佔領，波蘭地下國民軍與華沙市民發動的8月1日起義事件。這波起義長達63天，當時背景是蘇聯已經逐步壓制德軍，推到華沙城郊的維斯拉河。However，波蘭人也認為讓蘇聯軍進城根本是引虎驅狼，沙俄對波波的野心路人皆知。甚至沙俄曾經密謀想和德國一起瓜分波波。然而德軍仍然成功鎮壓華沙起義，也帶給華沙極大的傷害。

Warsaw Uprising 紀念碑

二戰末期，蘇聯軍隊迫使德軍節節敗退，很快迫近華沙。看到解放指日可待，華沙市民於1944年8月1日實行起義，抗擊德軍。起義的結果使得市中心一度得以解放。但蘇軍由於政治軍事上諸多原因，軍隊暫停駐在維斯瓦河對岸就不再前進了。起義由於失去援軍而不得不節節敗退。這次起義中有20萬人犧牲，城區幾乎全部被毀。10月2日，華沙投降了。

1989年8月，人們為了紀念華沙起義45週年建成了這座華沙起義紀念碑（Pomnik Powst.Warszwaskiego 1944 r.）。

Lazienkowski Park 蕭邦公園

華沙市民為傲的公園，佔地76公頃，走到累死不償命。裡面的特色大概就是有一座蕭邦的雕像以及水上宮殿。1764年由波波國王(Stanislaw August Poniatowski)所建，並在湖面上蓋一人工島及一宮殿(Palace on the water)。

Alaska - Aurora in Fairbanks & Polarbear in Barrow

Dec 1st, 2013

畢業之後一直都想去阿拉斯加看極光和北極熊，但是礙於鬼島的研發替代役只能出國8天，只好延後到服完役期，且剛好2012~2013是極光極大期。這次去阿拉斯加總共10天, 其中去Barrow看北極熊是9/24，當天從Fairbanks來回Barrow；剩下九天都在Fairbanks。

行前準備

9月的阿拉斯加其實不會冷到靠背，晚上頂多-5℃ ，把去合歡山賞雪的裝備帶一帶就好了…(Barrow除外，這真是可怕的地方，穿兩件褲子還是會從腳底冷上來)，以下是冬天的準備(參考):

  • 頭部：保暖的帽子（毛線帽或頭套）與圍巾（耳罩可視個人需要準備)護膚乳、護唇膏、太陽眼鏡。若有需要也請帶足量的個人適用常備藥物。

  • 手部與腳部：厚手套（操作相機時最好還要有薄手套）、棉襪與毛襪（多一層絲襪也能保暖）、登山鞋（底面能止滑，儘量選用防水與保暖的材質，若有多一點的空間還可塞暖暖包）。

  • 穿著：以「多層式」穿法為宜，因為車子與室內都有暖氣供應，最好能穿脫方便。衣服：長衛生衣褲、毛衣、棉質衣服、厚襯衫、厚羽絨衣外套（能防水最好）。褲子：棉質長褲、牛仔褲，外面若能再罩一件雙層的防風防水褲最好。

Activemerchant Allpay Plugin for Taiwan Payment

Nov 28th, 2013

ActiveMerchantAllpay

This plugin is an active_merchant patch forAllpay(歐付寶) online payment in Taiwan. Now it supports Credit card(信用卡), ATM(虛擬ATM) and CVS(超商繳費).

Installation

Add this line to your application’s Gemfile:

gem 'activemerchant'
gem 'active_merchant_allpay', '>=0.1.0'

And then execute:

$ bundle install

Or install it yourself as:

$ gem install activemerchant
$ gem install active_merchant_allpay
