Activemerchant Allpay Plugin for Taiwan Payment

Nov 28th, 2013

ActiveMerchantAllpay

This plugin is an active_merchant patch forAllpay(歐付寶) online payment in Taiwan. Now it supports Credit card(信用卡), ATM(虛擬ATM) and CVS(超商繳費).

Installation

Add this line to your application’s Gemfile:

gem 'activemerchant'
gem 'active_merchant_allpay'

And then execute:

$ bundle install

Or install it yourself as:

$ gem install activemerchant
$ gem install active_merchant_allpay

Usage
